Structure

This Template has a Responsive layout and is based on the Bootstrap Framework. The structure is very simple and clear.

If you would like to edit the color, font, or style of any elements, you can custom the template editing the CSS file (style.css).

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours.

HTML Structure

The HTML Structure is very simple and you can use the classes made available from Bootstrap Framework. If you need add one section create div class="section padding-top".

CSS Structure

The main css file is style.css and it contains all of the specific stylings for the page.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.
